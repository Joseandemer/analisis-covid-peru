# Análisis de Fallecidos por COVID-19 en Perú

Herramientas utilizadas:
- Python (pandas, numpy)
- Excel
- Power BI

Descripción:
Proyecto de análisis exploratorio de datos enfocado en el
estudio de fallecidos por COVID-19 en Perú, evaluando
variables como edad, año y patrones de mortalidad.
El objetivo es identificar tendencias y apoyar el análisis
estadístico de la información.
